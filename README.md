## 🚀 快速开始

### 环境要求
- Node.js >= 14
- npm >= 6

### 安装步骤

1. 克隆仓库

bash

git clone https://github.com/OpenPromise/Promise_finance

cd Promise_finance

2. 安装依赖

bash

npm install

3. 启动应用

bash

npm start

4. 构建应用

bash

npm run build


## 🛠️ 技术栈
